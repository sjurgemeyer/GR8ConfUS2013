# Upgrading Grails from 1.x to 2.x
This repo contains my reveal.js slides from my Gr8Conf 2013 talk on upgrade as well as the little bit of example code I had time to crank out for my talk.

I would happily welcome pull requests to add any example code you're able to contribute that shows changes required to a certain feature between versions, and if I have time I will try to add some myself.

Thanks to anyone who attended the talk or finds value in this repo!
